# FINANCIAL-DASHBOARD

#Description

Developed an interactive Revenue Dashboard to analyze total revenue, average revenue, transactions, and country performance across regions, products, quarters, and store rankings for data-driven business decisions.

#Technologies Used

Power BI Desktop

Power Query (ETL)

DAX (Revenue, Averages, KPIs)

Data Modeling

Excel / CSV data sources

Power BI Service

#Key Features

KPI cards: Total Revenue, Average Revenue, Transactions, Countries

Region-wise transaction analysis

Country-wise revenue comparison

Store ranking by total revenue

Quarter-wise revenue trend analysis

Product-wise revenue with store ranking

Sales representative performance table

Interactive slicers: Year, Quarter, Store Rank
